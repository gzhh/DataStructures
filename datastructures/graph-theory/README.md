# Graph theory
G=(V, E)

有向图：<Vi, Vj>

无向图：(Vi, Vj)

完全图: 任意两个顶点都有一条边相连接

概念：

1. 度
    
    入度
    
    出度
    
2. 带权图
3. 连通图
4. 生成树（最小生成树）
    
    n个顶点，有且只有n-1条边
    

存储：

邻接矩阵

邻接表

遍历：

深度 DFS

广度 BFS

应用：

最小生成树

最短路径（算法）

### 图的最小生成树

krushal 先小边、避开回路

prime 更新生成树到各顶点的距离

### 最短路径

Dijkstra 通过边实现松弛

### 有向无环图

AOV（没有回路）邻接表存储方式（拓扑排序）

AOE 图与关键路径
